# Python 読解力習得ガイド

> **目的**: Pythonコードを「書かずに読める」ようになるための教科書
> **対象**: プログラミング言語を一切知らない状態から、「コードの意味を人に説明できる」レベルまで
> **所要時間**: 約40時間（1日1時間×1ヶ月）
> **学習方法**: コードを書く必要なし。読んで理解するだけ

**インタラクティブ版**: https://fukukei23.github.io/python-reading-guide/
（コードを1行ずつ実行できる、変数の値が見える、用語の意味がポップアップで出る）

## 構成

| セクション | 内容 | ファイル数 | 所要時間 |
|-----------|------|-----------|---------|
| [01_basics](01_basics/) | プログラミングの言語共通概念 | 20ファイル | 20時間 |
| [02_python_syntax](02_python_syntax/) | Python固有の書き方 | 10ファイル | 8時間 |
| [03_error_messages](03_error_messages/) | エラーメッセージ読解 | 10ファイル | 5時間 |
| [04_my_code_examples](04_my_code_examples/) | 自分のリポジトリのコードを読む | 4ファイル | 7時間 |
| [05_quiz](05_quiz/) | 理解度確認クイズ | 3ファイル | - |

## 使い方

1. `01_basics/` から順番に読む
2. 各ファイルに「例題」と「確認問題」がある
3. 確認問題に答えられたら次に進む
4. 最後に `04_my_code_examples/` で実戦練習
5. `05_quiz/` で総合理解度を確認

## 例題の出典

この教科書の例題は、以下の自分のリポジトリから実際に使われているコードを引用しています：

- **NexusCore** — マルチエージェントAIフレームワーク（Python）
- **atelier-kyo-manager** — 物販管理システム（Python/Flask）
- **reserve-optimizer** — 予約管理LINE Bot（JavaScript/GAS）

## 到達目標

この教科書を完了すると：

- ✅ Pythonコードの各行が何をしているか説明できる
- ✅ エラーメッセージを見て「何が起きているか」が分かる
- ✅ AIに的確な指示（プロンプト）を出せる
- ✅ コードレビューで「ここは問題ない」「ここは怪しい」が判断できる
- ✅ 「Python（AI併用）で開発できます」と就活で言える
