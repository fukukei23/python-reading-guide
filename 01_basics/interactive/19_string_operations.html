19. 文字列操作(string operations)

CONCEPT

文字を組み立てたり、分割したりする方法です。LINEbotのメッセージ作成や、CSVデータの解析などでよく使います。

▶ コードを1行ずつ実行してみよう
example.pyPython
1order_id = "A001"
2total = 890
3message = f"注文番号: {order_id}, 合計: ¥{total:,}"
4csv_line = "田中,25,東京都"
5fields = csv_line.split(",")
→ « 次の行 » ボタンを押して実行を開始
VARIABLES
実行すると変数の状態が表示されます
📚 読み方ガイド
コード読み方
f"合計: ¥{total:,}"f-string: 変数を文字列に埋め込む。{total:,} はカンマ区切り
", ".join(items)リストの要素を「, 」でつなぐ
csv_line.split(",")文字列を「,」で分割してリストにする
text.strip()前後の余分な空白を削る
text.replace("A", "B")「A」を「B」に置き換える
KEY POINT

f"...{変数}..." で変数を文字列に埋め込む（f-string）。{value:,} でカンマ区切り、{value:.2%} でパーセント表示。split() で分割、join() で結合。strip() で空白削除。

❓ 確認問題
1このコードは何を表示しますか？
users = ["田中", "佐藤", "鈴木"] result = "、".join(users) print(result)
「田中、佐藤、鈴木」と表示されます。join() はリストの要素を指定した文字列（「、」）でつなぎます。
2formatted には何が入りますか？
price = 1980 tax = 1.1 total = int(price * tax) formatted = f"税込: ¥{total:,}"
formatted には "税込: ¥2,178" が入ります。1980 * 1.1 = 2178.0 → int で 2178 → カンマ区切りで 2,178 になります。
3このコードは何を表示しますか？
text = "banana,apple,cherry" fruits = text.split(",") print(fruits[1])
「apple」と表示されます。split(",") で ["banana", "apple", "cherry"] に分割され、インデックス1番目は "apple" です。
