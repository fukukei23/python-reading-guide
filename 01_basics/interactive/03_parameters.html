3. 引数と戻り値(parameter / return)

CONCEPT

引数（ひきすう）は「関数に渡す材料」、戻り値は「関数から戻ってくる結果」です。レシピに材料を渡して、完成品を受け取るイメージ。

▶ コードを1行ずつ実行してみよう
example.pyPython
1def add_tax(price):
2 return int(price * 1.10)
3
4total = add_tax(500)

実行すると変数の状態が表示されます
📚 読み方ガイド
コード読み方
def add_tax(price):price という材料を受け取る関数
return int(price * 1.10)price x 1.10 を整数にして返す
total = add_tax(500)500 を材料として渡し、結果を total に受け取る
msg = make_message("佐藤", "14:00")2つの材料（名前と時間）を渡す
KEY POINT

関数名の () の中に書くのが引数。呼び出し側の () の中に材料を書く。return の右側にあるものが戻り値。

❓ 確認問題
1result には何が入りますか？
def multiply(a, b):\n return a * b\n\nresult = multiply(3, 4)
result には 12 が入ります。multiply(3, 4) は 3 * 4 = 12 を返します。
2greet("鈴木") は何を返しますか？
def greet(name):\n greeting = name + "さん、いらっしゃい！"\n return greeting
"鈴木さん、いらっしゃい！" を返します。引数 name に "鈴木" が渡されます。
3a と b にはそれぞれ何が入りますか？
def shipping_cost(region, weight):\n if region == "海外":\n return weight * 500\n return weight * 100\n\na = shipping_cost("国内", 3)\nb = shipping_cost("海外", 3)
a = 300（3 x 100）、b = 1500（3 x 500）。"国内" は海外ではないので weight * 100、"海外" は条件に一致するので weight * 500 が返ります。
