# 2. f-string — 文字列に変数を埋め込む

## ざっくり言うと
文字列の中に `f"..."` と書くと、`{}` の中に変数や計算をそのまま入れられます。
「f」は **formatted（整形された）** の意味です。

## コード例

### 例1: 変数を埋め込む
```python
name = "たろう"
age = 25

print(f"私は{name}です。年齢は{age}歳です。")
# → 私はたろうです。年齢は25歳です。
```

### 例2: 計算を入れる
```python
price = 1200
tax = 0.1

print(f"税込み価格は{price * (1 + tax)}円です")
# → 税込み価格は1320.0円です
```

### 例3: 書式を整える
```python
pi = 3.14159

print(f"円周率は{pi:.2f}です")   # 小数点以下2桁
# → 円周率は3.14です

print(f"{'名前':<10}{'点数':>5}")  # 左寄せ・右寄せ
# → 名前          点数
```

## 読み方ガイド

1. 文字列の**開始引用符の前**に `f` をつける
2. 変数を入れたい場所に `{変数名}` と書く
3. `{ }` の中には**式（計算）**も書ける
4. `:.` の後に書式指定ができる（`.2f` = 小数2桁）

## よくあるパターン

- `f"{name}さん"` — あいさつ文を作る
- `f"{len(items)}個"` — 長さや個数を表示
- `f"{rate:.1%}"` — パーセント表示（0.85 → 85.0%）
- `f"{number:05d}"` — ゼロ埋め（42 → 00042）

## 確認問題

**問1**: 次のコードの出力は？
```python
fruit = "もも"
count = 3
print(f"{fruit}を{count}個買いました")
```

**問2**: f-stringを使わずに同じことをするには？（古い書き方）
```python
name = "花子"
# "花子さん、こんにちは" と表示したい
```

**問3**: 次のコードの出力は？
```python
x = 7
print(f"{x * 3}は{9 + 12}です")
```
