# 1. インデント（字下げ）— Pythonのブロック

## ざっくり言うと
Pythonでは、スペースの数で「ここからここまでがひとまとまり」を表します。
他の言語が `{ }` を使うところを、Pythonは**インデント（字下げ）**で表します。

## コード例

### 例1: if文のブロック
```python
age = 20

if age >= 18:
    print("大人です")      # ← ここは4スペース右にある
    print("投票できます")  # ← ここも同じ4スペース
print("終了")              # ← ここは外にある
```

### 例2: forループの中
```python
fruits = ["りんご", "みかん", "バナナ"]

for fruit in fruits:
    print(fruit)           # ← ループの中
    print("美味しい")      # ← ループの中
print("すべて食べた")      # ← ループの外
```

### 例3: ネスト（入れ子）
```python
scores = [80, 45, 90]

for score in scores:
    if score >= 60:
        print("合格")       # ← 8スペース（2段階右）
    else:
        print("不合格")     # ← 8スペース
```

## 読み方ガイド

1. **コロン `:` の次の行**は必ず右にずれる
2. **同じスペース数** = 同じグループ
3. ずれる量は**4スペース**がルール（タブは使わない）
4. スペースの数が違うと**エラーになる**

## よくあるパターン

- **`if` / `elif` / `else`** の中身はすべてインデントする
- **`for` / `while`** の中身はインデントする
- **`def`（関数定義）** の中身もインデントする
- インデントを戻すと「ブロックの終わり」を意味する

## 確認問題

```python

    print("こんにちは")
```

```

    print(x)

```
